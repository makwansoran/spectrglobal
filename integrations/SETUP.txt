Spectr Intelligence Platform — AI Hedge Fund backend
====================================================

This site connects to the open-source backend:
https://github.com/virattt/ai-hedge-fund

Quick start (from the hedge fund repo root)
-------------------------------------------
1. git clone https://github.com/virattt/ai-hedge-fund.git
2. cp .env.example .env   (add OPENAI_API_KEY and FINANCIAL_DATASETS_API_KEY)
3. ./run.sh               (Mac/Linux) or run.bat (Windows)

The API should listen on http://localhost:8000

On index.html, set "API base URL" to that address, or before scripts load:
  window.SPECTR_PLATFORM = { apiBase: "http://localhost:8000" };

Endpoints used by the Spectr frontend
-------------------------------------
GET  /hedge-fund/agents  (also used as health check)
GET  /
POST /hedge-fund/run   (Server-Sent Events stream)

CORS: the upstream backend allows localhost:5173 by default. If you serve
this static site from another origin (e.g. file:// or spectrglobal.vercel.app),
add your origin to app/backend/main.py allow_origins or run the API behind a
proxy on the same domain.
